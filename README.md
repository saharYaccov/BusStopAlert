# התראת קרבה לתחנת אוטובוס 🔔
קישור ישיר לאפליקציה -
[app](https://saharyaccov.github.io/BusStopAlert/)
אפליקציה חכמה שעוקבת אחר מיקומך ומתריעה אוטומטית כשאתה מתקרב לתחנת האוטובוס שלך.

## תכונות ✨

- 🔍 **חיפוש חכם** - חפש תחנות לפי שם (לא צריך קואורדינטות!)
- 📍 **מעקב רציף** - עוקב אחר מיקומך כל 5 שניות
- 🔔 **התראה אוטומטית** - צלצול וחלון קופץ כשאתה במרחק פחות מ-1.5 ק"מ
- 🗺️ **מפה אינטראקטיבית** - רואה את המיקום שלך והתחנה במפה
- 📏 **חישוב מרחק מדויק** - נוסחת Haversine
- ⏱️ **זמן הליכה משוער**
- 🎨 **עיצוב מודרני** עם אנימציות
- 📱 **תמיכה מלאה במובייל**

## דרישות 🛠️

### קבצים נדרשים:
1. `index.html` - דף HTML ראשי
2. `styles.css` - עיצוב
3. `script.js` - לוגיקה
4. **תיקיית `alert/`** - חובה! מכילה קובץ צלצול

### קובץ הצלצול:
צור תיקייה בשם `alert` ושים בה אחד מהקבצים הבאים:
- `a01.mp3` (מומלץ - נתמך בכל הדפדפנים)
- `a01.ogg` (אופציונלי)
- `a01.wav` (אופציונלי)

## איך להוסיף קובץ צלצול? 🔊

### אפשרות 1: הורד צלצול מוכן
1. עבור לאתר: https://freesound.org או https://mixkit.co/free-sound-effects/
2. חפש "alert sound" או "notification"
3. הורד קובץ MP3 קצר (2-5 שניות)
4. שנה את שם הקובץ ל-`a01.mp3`

### אפשרות 2: צור צלצול משלך
1. הקלט צלצול בטלפון או במחשב
2. המר לפורמט MP3 (באמצעות https://cloudconvert.com/wav-to-mp3)
3. שמור בשם `a01.mp3`

### אפשרות 3: השתמש בצלצול מובנה בדפדפן
אם אין לך קובץ, האפליקציה תציג התראה חזותית בלבד (ללא צלצול).

## התקנה והרצה 🚀

### העלאה ל-GitHub Pages:

1. **צור תיקייה** בשם `alert` ב-repository
2. **העלה את קובץ הצלצול** (`a01.mp3`) לתיקיית `alert`
3. **העלה את שאר הקבצים**:
   - `index.html`
   - `styles.css`
   - `script.js`
4. הכנס לכתובת: `https://saharyaccov.github.io/portfolio/BusStopAlert/`

### הרצה מקומית:

```bash
# עם Python
python -m http.server 8000

# עם Node.js
npx serve

# אז פתח: http://localhost:8000
```

## שימוש 📖

### שלב 1: אשר הרשאת מיקום
כשתפתח את האפליקציה, הדפדפן יבקש הרשאה לגישה למיקום - **חובה לאשר**.

### שלב 2: חפש תחנה
1. הקלד שם תחנה בשדה החיפוש, לדוגמה:
   - "תחנה מרכזית חיפה"
   - "רמב״ם חיפה"
   - "קניון לב חיפה"
2. לחץ "חפש תחנה"
3. בחר תחנה מהרשימה

### שלב 3: התחל מעקב
1. לחץ על "התחל מעקב"
2. האפליקציה תעקוב אחר מיקומך כל 5 שניות
3. כשתהיה במרחק של פחות מ-1.5 ק"מ - **תישמע התראה!** 🔔

### שלב 4: עצור מעקב
לחץ "עצור מעקב" כשהגעת לתחנה או כשרצית להפסיק.

## שאלות נפוצות ❓

**ש: האם האפליקציה תמשיך לעבוד כשהמסך כבוי?**
ת: תלוי בדפדפן ובמכשיר. רוב הדפדפנים מאטים או עוצרים JavaScript כשהאפליקציה ברקע.

**ש: האפליקציה צורכת סוללה?**
ת: כן, שימוש ב-GPS צורך סוללה. המעקב מתבצע כל 5 שניות כדי לחסוך בסוללה.

**ש: האם עובד בטיסה/במצב אופליין?**
ת: לא. דרושה חיבור לאינטרנט לטעינת המפה ולחיפוש תחנות.

**ש: למה הדפדפן מבקש הרשאה למיקום?**
ת: זה תקן אבטחה - אף אתר לא יכול לגשת למיקום בלי הרשאה מפורשת ממך.

**ש: האם אפשר להשתמש ביותר מתחנה אחת?**
ת: כרגע לא. חפש תחנה חדשה כדי לעקוב אחרי תחנה אחרת.
